1. PRÉSENTATION DE L'ACEEPCI
Identité
•	Nom complet : Association Chrétienne des Élèves et Étudiants Protestants de Côte d'Ivoire
•	Création : 1er novembre 1961
•	Siège : MAPE, Boulevard de l'Université, Abidjan-Cocody
•	Membres : 5 000+ actifs répartis en 88+ départements
•	Devise : Connaître — Aimer — Servir
•	Vision : Gagner l'école ivoirienne à Christ et former les leaders intellectuels de demain
•	Affiliation : FUACE, FEMAJECI, Église Méthodiste CI
Activités principales
•	Camp Biblique National annuel (57+ éditions)
•	Évangélisation en milieu scolaire et estudiantin
•	Formation spirituelle, leadership et gestion administrative
•	Chorales nationales, concours, expositions culturelles
•	Soutien social (bourses, aide directe)
2. CONTEXTE ET OBJECTIFS
Contexte
Après 60+ ans d'existence, l'ACEEPCI souhaite se doter d'une plateforme moderne pour améliorer sa communication, gérer ses membres et collecter des fonds en ligne de façon sécurisée.
Objectifs
•	Améliorer la visibilité et le rayonnement en ligne
•	Faciliter la communication entre le bureau national, départements et membres
•	Permettre adhésions, cotisations et dons en ligne (sécurisés)
•	Centraliser les inscriptions aux activités (camp, séminaires, formations)
•	Mobiliser la diaspora et partenaires internationaux
Public cible
•	Élèves/Étudiants : Membres actifs sur le territoire
•	Anciens membres : Alumni et diaspora
•	Donateurs : Particuliers, entreprises, partenaires
•	Parents/tuteurs : Intéressés par les activités
•	Visiteurs : Prospects souhaitant rejoindre l'association

5. STACK TECHNIQUE
Technologies principales
Composant	Solution
Frontend	Next.js 14+ (App Router)
Backend API	Laravel 11+ (API RESTful)
Base de données	PostgreSQL (ou MySQL)
Paiements mobiles CI	CinetPay API / Kkiapay
Paiement international	Stripe / PayPal
Hébergement	VPS (DigitalOcean, OVH) ou cloud (AWS, Vercel)
CDN	Cloudflare
Emails transactionnels	SendGrid / Mailchimp / Brevo
Stockage fichiers	AWS S3 / Cloudinary
SSL	Let's Encrypt (HTTPS obligatoire)

Architecture technique
Frontend (Next.js)
•	Pages publiques (SSG/ISR)
•	Espace membres (SSR/CSR)
•	Dashboard admin (CSR)
•	API Routes (pour webhooks)
Backend (Laravel)
•	API RESTful (JSON)
•	Authentification JWT/Sanctum
•	Gestion paiements (CinetPay, Stripe)
•	Notifications (email, SMS)
•	Jobs & Queues (Laravel Horizon)
•	Admin panel (Laravel Nova optionnel)
Base de données (PostgreSQL)
•	Users, Membres, Départements
•	Transactions, Dons, Cotisations
•	Contenus CMS (articles, événements)
•	Logs et audit trail
Performances et sécurité
•	Temps de chargement < 3s (connexion 3G/4G CI)
•	Responsive (mobile, tablette, desktop)
•	Disponibilité 99,5% minimum
•	HTTPS obligatoire (certificat SSL actif)
•	Passerelles paiement certifiées PCI-DSS
•	Authentification 2FA pour admin
•	Sauvegarde automatique quotidienne
•	Protection SQL injection, XSS, CSRF
•	Conformité RGPD / protection données personnelles
